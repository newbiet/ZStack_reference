# 启动云主机（本地存储）

* ### 描述

 云主机由Stopped状态变为Running

* ### 初始状态

 [Stopped](/Unit/VM/status.md)

* ### 前提条件

 1. 云主机运行在本地存储的物理机上

 2. 物理机处于enabled状态，connectted状态

* ### 参数

 

 | 资源 | 状态 |
 | ---
 | [云主机](/Unit/VM/README.md) | state:running | 
 | [物理机](/Unit/Host/README.md) | state:Enabled ; status:Connected |

* ### 结束状态

 [Running](/Unit/VM/status.md)

* ### 错误

 1. 物理机资源不足
